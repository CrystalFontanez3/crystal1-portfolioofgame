# Number Guessing Game

A simple command-line number guessing game written in Python.

## Features

- Random number from 1–100
- Unlimited guesses
- Counts attempts
- Input validation
- Play option again

## Requirements

- Python 3.x

## Run

```bash
python guessing_game.py
```
